# 影像資料集

## 內容

此資料集包含以下類型的檔案：

- `p1` 到 `p5` 資料夾，每個資料夾代表不同的姿勢類別。
- `.png` 影像檔案

**資料夾結構：**

```
dataset/
 ├─ p1/
 ├─ p2/
 ├─ p3/
 ├─ p4/
 ├─ p5/
```

檔案範例：

- `DSC09234.png`
- `DSC09122.png`
- `DSC09118.png`
- `DSC09123.png`
- `DSC09121.png`
